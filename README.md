# Sharp Library Management System (LMS)

A straightforward, fast, and lightweight Command Line Interface (CLI) application built entirely in plain Java. This system is designed to handle core library operations efficiently, utilizing multithreading to manage concurrent tasks like book borrowing.

## Features

* **Book Management:** Add, update, view, and remove books from the library inventory.
* **Borrowing & Returning:** Efficient check-out and check-in processes.
* **Concurrent Processing:** Handles multiple borrowing tasks concurrently using Java Threads (e.g., `BorrowTask`).
* **CLI Driven:** Simple, interactive command-line menus for fast navigation.
* **Zero Dependencies:** Built with pure Java—no external frameworks or libraries required.
